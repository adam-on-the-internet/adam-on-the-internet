I use the free tier of Heroku for hosting my web applications.

I use a Mongo Database to store my data. I use a cloud database with Mongo Atlas.

For styling most of my applications, I use Bootstrap 4 and the Font Awesome version that it includes.

I use the free tier of Travis CI to automatically test most of my applications before they deploy.
I use GitHub for remote repositories for my code.
Most of my UIs are created using Angular.

Node w/ Express powers my API, connecting to my MongoDB via Mongoose. I also use a simple Express app to serve my Angular apps with Heroku.

I use the free application Metro Studio to generate Favicons for my applications.
